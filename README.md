# Book Class (C#)

Учебный проект на C#.

## Задание

Создать класс **Book** с полями:
- Название
- Автор
- Год выпуска

Создать список из 3 книг и вывести информацию в формате:
**Автор - Название (Год)**

## Описание решения

В программе:
- создан класс `Book`
- добавлен конструктор для удобного создания объектов
- используется список `List<Book>`
- вывод реализован через метод `PrintInfo()`

## Пример вывода

Фёдор Достоевский - Преступление и наказание (1866)  
Михаил Булгаков - Мастер и Маргарита (1967)  
Лев Толстой - Война и мир (1869)
